# Super-Sales-Dashboard-in-Power-BI-Excel
This project analyzes sales and inventory data for Adventure Works, a global retailer,  Through an interactive dashboard in Power BI and Excel
It enables users to track performance, analyze sales trends, monitor inventory levels, 
And make data-driven decisions to optimize business operations.
- Tools and Technologies: Data transformation, cleansing, and modeling with Power Query, built an interactive dashboard using Power BI and Excel, leveraging DAX for advanced calculations and insights
